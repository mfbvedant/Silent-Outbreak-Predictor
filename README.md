# Silent Outbreak Predictor

A monorepo for the Silent Outbreak Predictor application.

## Project Structure

This monorepo is organized into the following directories:

*   **[ai_core](file:///c:/Users/VEDAN/Silent-Outbreak-Predictor/ai_core)**: Contains the CrewAI multi-agent system responsible for outbreak analysis and predictions.
*   **[backend](file:///c:/Users/VEDAN/Silent-Outbreak-Predictor/backend)**: Contains the FastAPI routing and server logic, optimized for deployment on Render.
*   **[frontend](file:///c:/Users/VEDAN/Silent-Outbreak-Predictor/frontend)**: Contains the user dashboard and frontend application.
