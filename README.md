# Silent Outbreak Predictor

A monorepo for the Silent Outbreak Predictor application.

## Project Structure

This monorepo is organized into the following directories:

*   **[ai_core](file:///c:/Users/VEDAN/Silent-Outbreak-Predictor/ai_core)**: Runs the CrewAI pipeline for multi-agent outbreak forecasting.
*   **[backend](file:///c:/Users/VEDAN/Silent-Outbreak-Predictor/backend)**: Handles FastAPI routing and async file serving.
*   **[frontend](file:///c:/Users/VEDAN/Silent-Outbreak-Predictor/frontend)**: The Streamlit dashboard user interface.
*   **[data_outputs](file:///c:/Users/VEDAN/Silent-Outbreak-Predictor/data_outputs)**: Shared directory containing generated outputs, reports, and logs.
